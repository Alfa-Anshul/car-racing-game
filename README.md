# 🏎️ Car Racing Game

Full-stack 3D car racing game with WASD controls, leaderboard, profiles, and more.

## Stack
- **Frontend**: React + Three.js (3D WebGL)
- **Backend**: FastAPI + PostgreSQL
- **Auth**: JWT tokens
- **Deploy**: Docker + Nginx
- **Domain**: https://car.anervea.live

## Features
- 🎮 Multi-car 3D racing with WASD controls
- 🔐 Full authentication (register/login)
- 🏆 Global leaderboard
- 👤 Racing profile with stats
- 📊 Race history & scores
- 🎨 Shader-based graphics (GLSL)
- 🤖 AI opponent cars

## Run Locally
```bash
docker-compose up --build
```
